# Customer Relationship Management (CRM) – Customer Lifetime Value Prediction

- Project Overview

This project focuses on building a Customer Relationship Management (CRM) system that predicts Customer Lifetime Value (CLV) using machine learning. CLV helps businesses understand the future value of customers and make better marketing and retention decisions.

# Workflow

1. Data Preprocessing

Data cleaning

Handling missing values

Data validation



2. Feature Engineering

Recency (last purchase time)

Frequency (number of purchases)

Monetary value (total spending)



3. Machine Learning Models

BG/NBD model for purchase prediction

Gamma-Gamma model for spending prediction

CLV prediction using both models



4. Experiment Tracking

MLflow for logging experiments, models, and metrics



5. CI/CD Pipeline

GitHub for version control

Jenkins for automated pipeline



6. Deployment

Flask API for serving the model

Cloud deployment for production use


- Technologies Used

Python

MLflow

Flask

GitHub

Jenkins

Machine Learning (BG/NBD, Gamma-Gamma)


- Outcome

The system predicts Customer Lifetime Value, helping businesses identify high-value customers and improve CRM strategies.
